# Integracion con Code - OSS

Este directorio esta reservado para tu fork de `microsoft/vscode`.

## Recomendacion

Usa este flujo:

1. Crea un fork de `https://github.com/microsoft/vscode`.
2. Clonalo dentro de este directorio.
3. Mantiene tus cambios de branding y distribucion en ramas propias.
4. Conserva toda la capa AI en `extensions/` y `packages/`.

## Cambios que suelen vivir aqui

- nombre del producto
- iconos
- `product.json`
- endpoints propios
- configuracion de actualizacion
- defaults de extensiones y ajustes
